What is SmartNotes?

SmartNotes is a small web application to take notes of the things you need to remember on your day-to-day basis.

How to use SmartNotes?

SmartNotes has an intuitive User Interface.

  • As soon as the webpage loads, the user will be presented with a set of notes that he has entered previously.
  • The user is given an option to,
    • Add a new note.
    • Delete existing notes.
    • Find notes similar to a particluar note.
    • Search for notes based on any keyword.
  • Adding a note requires the user to type in his note in a region provided as shown below

    and click on save.
  • As soon as the note is saved, it gets two more options,
    • Delete
    • Similar

    The delete option deletes that particular note.
    The similar option finds a set of notes similar to the clicked note as shown below,



    and shows the similarity value as well in the corner of the each note with a red background.
    Jaccard Similarity is the method used to measure the similarity in this application.
    Jaccard Similarity can be represented mathematically as follows, 

    similarity_value = intersection(termListA, termListB) / union(termListA, termListB)
    where,
    termListA is the term list of note A
    termListB is the term list of note B
    similarity_value is the final similarity value evaluated by the formula
  • The search bar at the top right allows the user to search for a note in the set of notes the user
    has previously saved. A relevance score is shown at the top-right corner of the notes that appear
    in the search results as shown below.



    The relevance score is computed as the ratio of number of terms in the query by
    number of terms in query that are actually matched in the note. It can be represented mathematically as follows, 

    relevance_score = (number of words in query) / (number of words found in the note)

How was this built?

This intelligent web application was built using the following technologies,

  • Python2.7 programming language
  • MongoDB, a scalable non-relational database
  • Flask web application framework
  • HTML5 and CSS3 via initializr.com
  • Couple of third party python libraries like nltk (Natural Language Toolkit), Pymongo, etc.
