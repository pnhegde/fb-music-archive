What is this?

Music Archive.

How was this built?

This web application was built using following technologies,

  • Python 2.7 programming language
  • Flask web application framework
  • MongoDB, a scalable non-relational database
  • HTML5 and CSS3 via initializr.com
  • Third party python libraries like nltk (Natural Language Toolkit), Pymongo, Facebook Graph APIs, LXML etc.
